# AI ニュースダイジェスト — 2026-03-29

## 1. OpenAI、GPT-5.4 をリリース — 推論・コーディング・エージェントを統合

- **URL**: https://note.com/masao_n/n/n28bdde1b23f3
- GPT-5.4 がリリースされ、1M トークンコンテキストやネイティブコンピュータ操作を搭載。
- GPT-5.2 比で個別主張エラー率 33%削減、レスポンス全体エラー率 18%削減を実現。
- 同時に GPT-5.1 シリーズは 3 月 11 日に廃止され、GPT-5.3/5.4 へ自動移行が完了した。

## 2. OpenAI、動画生成 AI「Sora」の全面終了を発表

- **URL**: https://note.com/hiroe28/n/n478c11aedc7c
- 3 月 24 日、OpenAI は動画生成 AI「Sora」のアプリ・API・ChatGPT 内機能をすべて終了。
- 1 日あたり約 1,500 万ドルの計算コストが経営を圧迫し、リリースからわずか半年での撤退。
- 日本のアニメ業界からの著作権侵害批判やディープフェイク問題も撤退の要因となった。

## 3. Claude Opus 4.6 が 1M コンテキストで一般提供開始

- **URL**: https://www.ai-souken.com/article/claude-code-updates-2026
- 3 月 13 日、Anthropic の Claude Opus 4.6 が 100 万トークンコンテキストに正式対応。
- 日本語で約 50 万文字（新書 4〜5 冊分）を一度に読み込ませることが可能に。
- 追加料金なしで利用可能で、旧モデル（Opus 4/4.1）からの自動移行も完了した。

## 4. Claude Code に音声モード・Computer Use・Channels が追加

- **URL**: https://note.com/kawaidesign/n/n2407bba0ae03
- 3 月 3 日に `/voice` コマンドで音声操作が可能に（プッシュトゥトーク方式、20 言語対応）。

- 3 月 19 日の「Channels」により Discord や Telegram から Claude Code へメッセージ送信が可能に。

## 6. 英語版 Wikipedia、LLM による記事作成を原則禁止
