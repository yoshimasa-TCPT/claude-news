## 1. 国連「AIガバナンスに関するグローバル対話」本日ジュネーブで開幕

- **URL:** https://news.un.org/en/story/2026/07/1167848
- **要約:**

## 2. Claude Sonnet 5がFree/Proユーザーのデフォルトモデルに — Opus 4.8に迫る性能を低価格で提供

- **URL:** https://www.watch.impress.co.jp/docs/news/2121365.html
- **要約:**
  - Anthropicが6月30日にClaude Sonnet 5を発表し、7月1日からClaude Free/Proの全ユーザーにデフォルトモデルとして提供開始。Claude Code、API、AWS Bedrock、Google Cloud等で同時利用可能。

## 4. GitHub Models、7月30日に完全廃止 — Azure AI Foundryへの移行推奨

- **URL:** https://github.blog/changelog/2026-07-01-github-models-is-being-fully-retired-on-july-30-2026/
- **要約:**
  - GitHubがGitHub Modelsの7月30日完全廃止を正式発表。プレイグラウンド、モデルカタログ、推論API、BYOKを含む全機能が停止される。
  - 7月16日と23日にブラウンアウト（短期的なサービス停止）を実施予定。その間リクエストは一時的にエラーを返す。
  - 代替としてAzure AI Foundry、またはGitHub上でのAI統合にはGitHub Copilotの利用を推奨。新規ユーザーへの提供は6月16日に既に終了済み。
