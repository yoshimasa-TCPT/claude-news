# AI ニュースダイジェスト - 2026年5月29日

## 1. Anthropic、「ミュトス級」AIを数週間以内に一般公開へ

**URL:** https://www.nikkei.com/article/DGXZQOGN28D7M0Y6A520C2000000/

Anthropicが5月28日、非公開の最先端AIモデル「Claude Mythos」と同等性能のモデルを数週間以内に全顧客へ提供すると発表した。
Mythosは2026年4月7日に発表された超高性能モデルで、サイバーセキュリティ分野で突出した能力を持ち、これまで信頼できるパートナー企業のみに限定提供されていた。
日本を含む世界各国で利用可能になる見込みで、3メガ銀行（MUFG・三井住友・みずほ）も5月末までにアクセス権を取得予定。

## 2. Claude Opus 4.8 リリース — コーディング性能向上・Dynamic Workflows対応

**URL:** https://blog.serverworks.co.jp/2026/05/29/060000

Anthropicが5月28日、フラッグシップモデル「Claude Opus 4.8」をリリース。前モデルOpus 4.7からわずか41日での更新となった。
SWE-Bench Proで69.2%（4.7の64.3%から改善）を記録し、エージェント的コーディング・推論・財務分析の性能が向上。新機能「Dynamic Workflows」と「Effort control」も同時リリース。
API料金は入力$5/出力$25（100万トークンあたり）で据え置き。Fast modeは従来の3分の1の価格に引き下げ。

## 3. OpenAI、GPT-5.5 InstantをChatGPTの新デフォルトモデルに

**URL:** https://www.itmedia.co.jp/aiplus/articles/2605/06/news022.html

OpenAIが5月5日、GPT-5.5 Instantを全ChatGPTユーザー向けの新デフォルトモデルとして展開開始。GPT-5.3 Instantを置き換える。
高リスク領域での誤った主張を52.5%削減し、より正確・簡潔・自然な回答を実現。メモリソース機能により、回答のパーソナライズに使用された文脈をユーザーが確認可能に。
有料ユーザーは設定から3カ月間旧モデルの継続利用が可能。

## 4. ChatGPT for Excel / Google Sheets がグローバル展開

**URL:** https://openai.com/index/chatgpt-for-excel/

OpenAIが5月6日、ChatGPTをExcelとGoogle Sheetsのサイドバーに統合する機能をグローバル展開。GPT-5.5を活用したスプレッドシートネイティブなAI体験を提供。
数式の生成・修正、データ分析、シート構造の理解など、スプレッドシート作業を直接支援。Microsoft AppSourceやGoogle Workspace Marketplaceからアドインとして導入可能。
Business・Enterprise・Edu・K-12に加え、Free・Go・Pro・Plusの全プランで利用可能。

## 5. 日本3メガ銀行、Claude Mythosのアクセス権を取得 — 国内民間企業初

**URL:** https://aifriends.jp/japan-megabanks-anthropic-claude-mythos-access-2026/

5月13日、MUFG・三井住友・みずほの3メガバンクがClaude Mythosのアクセス権を取得することが報じられた。日本の民間企業として初の事例。
Anthropicの信頼パートナープログラム「Project Glasswing」に参加する金融機関・セキュリティ企業のみに限定提供されていたモデルへのアクセスとなる。
5月末までにアクセス開始予定で、金融業務でのAI活用が新たな段階に入る。

## 6. NEC「AI Platform Service」ソフトウェア版を5月末より提供開始

**URL:** https://jpn.nec.com/press/202604/20260424_01.html

NECが100以上のサービス機能群を集約した「AI Platform Service」のソフトウェア版を5月末より提供開始。AIaaS版は7月から順次提供予定。
NEC独自の40以上の機能とグローバルパートナーの先端技術を垂直統合し、マルチクラウド・オンプレミス環境での展開を支援する。
AI活用に必要なフルスタック機能（アプリ/AIエージェント、モデル連携、データ統合）をワンストップで提供し、企業のAX（AI Transformation）を加速。

## 7. JBS、Varonis Systems Japanと生成AIデータセキュリティで連携開始

**URL:** https://www.nikkei.com/nkd/disclosure/tdnr/20260526547966/

日本ビジネスシステムズ（JBS）が5月29日、生成AI活用拡大に伴うデータセキュリティ課題に対応するため、Varonis Systems Japanとの連携開始を発表。
機密データの可視化、アクセス権管理、AIセキュリティの3領域で協力し、生成AI普及に伴う新たなリスクに対応する。
JBSのMicrosoft Cloud運用ノウハウとVaronisのデータセキュリティ基盤を組み合わせ、企業のデータガバナンス強化を支援。

## 8. Claude Code v2.1.143 アップデート — プラグイン・バックグラウンドセッション改善

**URL:** https://note.com/kazu_t/n/nef4729e12306

5月15日リリースのClaude Code v2.1.143で、プラグインとバックグラウンドセッション関連の改善が多数実施された。
Fast modeのデフォルトモデルがOpus 4.7に変更され、/goalコマンドの追加やclaude agentsのResearch Previewも登場。
Claude Codeは単なるコーディング支援ツールから、複数タスクを並行管理し完了条件まで自律的に進める「agentic coding環境」へと進化しつつある。
