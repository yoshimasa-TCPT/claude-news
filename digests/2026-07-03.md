# AI ニュースダイジェスト — 2026年7月3日

## 1. Anthropic、科学研究向けAIワークベンチ「Claude Science」をパブリックベータで公開

- **URL:** https://gigazine.net/news/20260701-anthropic-claude-science/
- **要約:**
  - Anthropicが6月30日、科学者向けAIワークベンチ「Claude Science」をパブリックベータとして公開。ゲノミクス・プロテオミクス・構造生物学・ケモインフォマティクスなど60以上の科学データベースとツールキットを統合。
  - 新モデルではなく既存のClaude Opus 4.8上で動作し、研究室のインフラ上で処理可能なためデータプライバシーが確保される。Pro/Max/Team/Enterpriseプランで利用可能。
  - 最大3万ドルの研究助成金プログラムも同時発表。50プロジェクトを対象に7月15日まで応募受付中で、9月〜12月に実施予定。

## 2. OpenAI、GPT-5.6シリーズ（Sol/Terra/Luna）の限定プレビューを開始

- **URL:** https://openai.com/index/previewing-gpt-5-6-sol/
- **要約:**
  - OpenAIが6月26日に次世代モデル群「GPT-5.6」を発表。最難問題向けのSol、業務向けのTerra、高速・低価格のLunaの3モデル構成で限定プレビューを開始。
  - 料金はSolが入力$5/出力$30、Terraが$2.50/$15、Lunaが$1/$6（100万トークンあたり）。SolはTerminal-Bench 2.1でコーディング最高スコアを記録。
  - 現在は約20社の信頼できるパートナー向けAPI・Codex経由の限定提供。ChatGPTでの一般提供は7月上旬〜中旬の見込み。7月にはCerebras上で750トークン/秒の高速推論も開始予定。

## 3. Claude Code v2.1.199リリース — バックグラウンドエージェントの安定性を大幅改善

- **URL:** https://dev.classmethod.jp/articles/20260703-cc-updates-v2-1-199/
- **要約:**
  - Anthropicが7月2日、Claude Code v2.1.199をリリース。バックグラウンドエージェントとサブエージェントの安定性、レート制限・ストリーミング・SSLなどネットワーク周辺の信頼性を中心に改善。
  - 環境変数CLAUDE_CODE_RETRY_WATCHDOGによる一時的エラーのデフォルトリトライ回数を300に引き上げ、MAX_RETRIESの上限（15）を撤廃。
  - Linuxでバックグラウンドエージェントデーモンが自身を強制終了する不具合、サブエージェントがAPIエラーを成功として報告する不具合などを修正。

## 5. Claude Opus 4.7のFast Mode非推奨化 — 7月24日に削除、Opus 4.8へ移行推奨

- **URL:** https://platform.claude.com/docs/ja/release-notes/overview
- **要約:**
  - AnthropicがClaude Opus 4.7のFast Modeを非推奨とし、2026年7月24日に完全削除予定と発表。Claude Opus 4.8のFast Modeへの移行が推奨されている。

## 7. PwC Japan、2026年生成AI技術動向レポートを公開 — 「オーグメンテッド・エンタープライズ」への転換を提唱

- **URL:** https://www.pwc.com/jp/ja/knowledge/thoughtleadership/generative-ai-survey2026-consideration.html
- **要約:**
  - PwC Japanが2026年の生成AI技術動向レポートを公開。AIエージェントが人間の思考と実行を同時に拡張し、複数エージェントが24時間並走してタスクを遂行する段階に入ったと分析。
  - 「オーグメンテッド・エンタープライズ」（AI拡張型企業）への転換を提唱。生成AIが「試す技術」から「業務に組み込まれる技術」へ完全に移行する転換点と位置づけ。
  - 一方でForresterは「エージェント機能を本格稼働している企業は15%未満」と慎重な採用予測も示しており、方向性は確定したが主流化にはまだ時間を要する状況。

## 8. 暗号通貨取引所OKX、AIエージェント経済のインフラ「OKX.AI」を発表

- **URL:** https://note.com/cotapon/n/n3c60aff4d146
- **要約:**
  - 暗号通貨取引所OKXが、AIエージェントが自律的に互いを「雇用」し「支払い」を行うことができる「エージェント経済」のためのインフラ「OKX.AI」を発表。
  - AIエージェント同士がタスクの委託・報酬の支払いを暗号通貨で自動処理する仕組みを構築。ブロックチェーンとAIエージェントの融合事例として注目される。
  - 自律型AIエージェントの経済活動を支える基盤技術として、Web3とAIの交差点における新たなユースケースを提示。
