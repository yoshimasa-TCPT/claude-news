# AI関連ニュースダイジェスト - 2026年3月27日

## 1. OpenAI GPT-5.4リリース、GPT-5.1は提供終了

- **URL**: https://note.com/masao_n/n/n28bdde1b23f3
- **要約**:
  - OpenAIが推論・コーディング・エージェントを統合した新モデル「GPT-5.4」をリリース。1Mトークンコンテキストとネイティブコンピュータ操作を搭載。
  - GPT-5.2比で個別主張エラー率33%削減、レスポンス全体エラー率18%削減を達成。
  - 3月11日にGPT-5.1シリーズは提供終了、既存ユーザーはGPT-5.3 Instant/GPT-5.4へ自動移行。

## 2. Claude Opus 4.6の1Mコンテキストが一般提供開始

- **URL**: https://www.ai-souken.com/article/claude-code-updates-2026
- **要約**:
  - 3月13日、Claude Opus 4.6の100万トークンコンテキストが追加料金なしで一般提供開始。
  - 日本語で約50万文字（新書4〜5冊分）を一度に処理可能に。
  - Opus 4/4.1のサポート終了に伴い、Opus 4.6へ自動移行が実施された。

## 3. Claude Codeに「Computer Use」機能が追加

- **URL**: https://note.com/fujin_metaverse/n/n2ad939c18794
- **要約**:
  - 3月23日、Claude CodeにAIがPCを直接操作する「Computer Use」機能が追加。
  - マウス操作、キーボード入力、ブラウザ操作などをAIが自律的に実行可能。
  - 音声モード（/voiceコマンド）やChannels機能（Discord/Telegram連携）も3月に追加された。

- **要約**:
