# AI関連ニュースダイジェスト - 2026年3月27日

## 1. OpenAI GPT-5.4リリース、GPT-5.1は提供終了

- **URL**: https://note.com/masao_n/n/n28bdde1b23f3
- **要約**:
  - OpenAIが推論・コーディング・エージェントを統合した新モデル「GPT-5.4」をリリース。1Mトークンコンテキストやネイティブコンピュータ操作を搭載。
  - 3月11日をもってGPT-5.1シリーズ（Instant/Thinking/Pro）はChatGPTで利用不可に。
  - 既存ユーザーはGPT-5.3 InstantまたはGPT-5.4へ自動移行された。

- **URL**: https://www.ai-souken.com/article/claude-code-updates-2026
- **要約**:
  - 3月13日、Claude Opus 4.6の100万トークンコンテキストが追加料金なしで一般提供開始。
  - Opus 4およびOpus 4.1のサポートが終了し、Opus 4.6へ自動移行。
  - Claude Sonnet 4.6が全プランのデフォルトモデルとして標準搭載されている。

## 3. Claude Codeに「Computer Use」機能が追加

- **URL**: https://note.com/fujin_metaverse/n/n2ad939c18794
- **要約**:
  - 3月23日、Claude CodeにAIがPCを直接操作する「Computer Use」機能が追加。
  - マウス操作、キーボード入力、ブラウザ操作などをAIが自律的に実行可能に。
  - オートメモリ、リモートコントロール、スケジュールタスクなどの新機能も同時期に追加。

## 6. 日本政府がAIエージェント・フィジカルAIに対応する指針を策定

- **URL**: https://www.nikkei.com/article/DGXZQOUA136YP0T10C26A2000000/
- **要約**:
  - 政府が自律的に動く「AIエージェント」やロボットを制御する「フィジカルAI」に対応するAI指針案をまとめた。
  - 誤作動やプライバシー侵害リスクを念頭に「人間の判断を必須とする仕組み」づくりを開発企業に求める。
  - AIの自律性が高まる中、安全性とガバナンスの両立を目指す内容。
